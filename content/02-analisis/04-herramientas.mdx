## Exponenciales y Logaritmos

<Definition title="Exponencial (definiciĂłn inductiva)">
Para $a \in \mathbb{R}$ y $n \in \mathbb{N}$:

1. $a^0 = 1$
2. $a^{n+1} = a^n \cdot a$, $n \ge 0$
3. $a^{-n} = 1 / a^n$
</Definition>

<Definition title="Exponencial (extensiĂłn a racionales)">
Para $a > 0$ y $m,n \in \mathbb{Z}, n \neq 0$:
$$a^{m/n} = \sqrt[n]{a^m}$$
</Definition>

<Theorem title="Propiedades del exponente racional">
Para $a,b > 0$ y $r,s \in \mathbb{Q}$:

1. $a^r a^s = a^{r+s}$
2. $(a^r)^s = a^{rs}$
3. $a^r / a^s = a^{r-s}$
4. $(ab)^r = a^r b^r$
5. $a^1 = a$, $a^0 = 1$
</Theorem>

<Definition title="Exponencial (exponente real)">
Para $a > 0$ y $r \in \mathbb{R}$:
$$a^r = \sup\{a^q : q \in \mathbb{Q}, q < r\} = \inf\{a^q : q \in \mathbb{Q}, q > r\}$$
</Definition>

<Definition title="El nĂşmero e">
$$e = \lim_{n \to \infty} \left(1 + \frac{1}{n}\right)^n$$
donde $2 < e < 3$
</Definition>

<Definition title="$(1+1/n)^n$ es estrictamente creciente" label="ProposiciĂłn">
La sucesiĂłn $a_n = \left(1 + \dfrac{1}{n}\right)^n$ es estrictamente creciente.

<Proof collapsible>
Por la desigualdad de medias aritmĂŠtica-geomĂŠtrica aplicada a $n+1$ nĂşmeros donde $n$ de ellos son $\frac{n+1}{n}$ y uno es $1$:
$$\left(\frac{n+1}{n}\right)^n \cdot 1 \leq \left(\frac{n \cdot \frac{n+1}{n} + 1}{n+1}\right)^{n+1} = \left(\frac{n+2}{n+1}\right)^{n+1}$$
Multiplicando ambos lados por $\left(\frac{n+1}{n}\right)^n$, se obtiene $a_n < a_{n+1}$.

Alternativamente, usando el Teorema del Binomio de Newton:
$$a_n = \sum_{k=0}^{n} \binom{n}{k} \frac{1}{n^k} = \sum_{k=0}^{n} \frac{1}{k!} \cdot \frac{n(n-1)\cdots(n-k+1)}{n^k}$$
Cada tĂŠrmino $\frac{n(n-1)\cdots(n-k+1)}{n^k} = \left(1-\frac{1}{n}\right)\cdots\left(1-\frac{k-1}{n}\right)$ es creciente en $n$, y $a_{n+1}$ tiene un tĂŠrmino adicional positivo, por lo tanto $a_n < a_{n+1}$.
</Proof>
</Definition>

<Definition title="Logaritmo natural">
Para $y > 0$:
$$\log(y) = \sup\{x \in \mathbb{R} : e^x < y\}$$
Es la funciĂłn inversa de $e^x$.
</Definition>

<Theorem title="Propiedades del logaritmo">
Para $a,b > 0$:

1. $\log(ab) = \log(a) + \log(b)$
2. $\log(a^b) = b \log(a)$

<Proof title="DemostraciĂłn de $\\log(xy) = \\log x + \\log y$" collapsible>
Sean $r = \log x$ y $s = \log y$, es decir $x = e^r$ y $y = e^s$. Entonces:
$$xy = e^r \cdot e^s = e^{r+s}$$
Aplicando $\log$ a ambos lados: $\log(xy) = r + s = \log x + \log y$.
</Proof>
</Theorem>

## Desigualdades Notables

<Theorem title="Desigualdad de Bernoulli">
Para $h > -1$ y $n \in \mathbb{N}$:
$$(1 + h)^n \ge 1 + nh$$
</Theorem>

<Theorem title="Desigualdad de Cauchy-Schwarz">
Sean $a_1, a_2, \dots, a_n$ y $b_1, b_2, \dots, b_n$ nĂşmeros reales:
$$(\sum a_i b_i)^2 \le (\sum a_i^2)(\sum b_i^2)$$
</Theorem>

<Theorem title="Desigualdad de la media aritmĂŠtica y geomĂŠtrica (AM-GM)">
Para $a_1, a_2, \dots, a_n > 0$:
$$\frac{a_1 + a_2 + \dots + a_n}{n} \ge \sqrt[n]{a_1 a_2 \dots a_n}$$
En particular, para dos nĂşmeros:
$$\frac{a + b}{2} \ge \sqrt{ab}$$
</Theorem>

<Theorem title="Desigualdad AM-GM de cuadrados">
Para $a, b \in \mathbb{R}$:
$$\frac{a^2 + b^2}{2} \ge \sqrt{a^2 b^2} = |ab|$$
</Theorem>

<Theorem title="VariaciĂłn de la desigualdad triangular">
Para $x, y \in \mathbb{R}$:
$$|x| = |x - y + y| \le |x - y| + |y|$$
*TĂŠcnica estĂĄndar en demostraciones $\varepsilon$-$\delta$: acotar $|f(x) - L| \le |f(x) - g(x)| + |g(x) - L|$.*
</Theorem>

<Theorem title="Teorema de Taylor">
Sea $f:I \to \mathbb{R}$ una funciĂłn $n$ veces derivable en un intervalo $I$ y sea $a \in I$. Entonces para todo $x \in I$ existe $c$ entre $a$ y $x$ tal que:
$$f(x) = \sum_{k=0}^{n-1} \frac{f^{(k)}(a)}{k!}(x-a)^k + \frac{f^{(n)}(c)}{n!}(x-a)^n$$
</Theorem>

## Ălgebra y Herramientas Combinatorias

<Theorem title="Diferencia de cuadrados">
Para $a, b \in \mathbb{R}$: $a^2 - b^2 = (a-b)(a+b)$.
</Theorem>

<Theorem title="Binomio de Newton">
Para $a, b \in \mathbb{R}$ y $m \in \mathbb{N}$:
$$(a + b)^m = \sum_{i=0}^{m} \binom{m}{i} a^{m-i} b^i$$
</Theorem>

<Theorem title="FactorizaciĂłn de diferencia de potencias">
Para $a, b \in \mathbb{R}$ y $k \in \mathbb{N}$:
$$a^k - b^k = (a - b)\sum_{i=0}^{k-1} a^{k-1-i}\, b^i$$
</Theorem>

<Theorem title="Serie geomĂŠtrica finita">
Para $a \ne 1$ y $n \in \mathbb{N}$:
$$1 + a + a^2 + \cdots + a^n = \frac{a^{n+1} - 1}{a - 1}$$
</Theorem>

<Theorem title="Serie aritmĂŠtica">
$$1 + 2 + \cdots + n = \frac{n(n+1)}{2}$$
</Theorem>

<Theorem title="TamaĂąo del conjunto potencia">
Para $n \in \mathbb{N}$:
$$\sum_{k=0}^{n} \binom{n}{k} = 2^n$$
</Theorem>

## TrigonometrĂ­a

<Theorem title="Desigualdad trigonomĂŠtrica">
Para $x \in (0, \pi/2)$:
$$\sin x \le x \le \tan x$$
Esta propiedad es clave para probar que $\displaystyle \lim_{x \to 0} \frac{\sin x}{x} = 1$.

<Proof collapsible>
Comparando ĂĄreas en el cĂ­rculo unitario con ĂĄngulo $x$:

<img src="/img/analisis/desigualdad-trigonometrica.svg" alt="" className="mx-auto" style={{width: '55%'}} />

- **TriĂĄngulo $OAP$:** vĂŠrtices en $O=(0,0)$, $A=(1,0)$, $P=(\cos x, \sin x)$. Ărea $= \tfrac{1}{2}\sin x$.
- **Sector circular $OAP$:** Ărea $= \tfrac{x}{2}$.
- **TriĂĄngulo $OAT$:** vĂŠrtices en $O$, $A=(1,0)$, $T=(1, \tan x)$. Ărea $= \tfrac{1}{2}\tan x$.

Como triĂĄngulo $OAP \subset$ sector $\subset$ triĂĄngulo $OAT$:
$$\frac{1}{2}\sin x \;\le\; \frac{x}{2} \;\le\; \frac{1}{2}\tan x$$

Multiplicando por $2$ y dividiendo por $\sin x > 0$:
$$1 \;\le\; \frac{x}{\sin x} \;\le\; \frac{1}{\cos x}$$

Tomando recĂ­procos (la desigualdad se invierte):
$$\cos x \;\le\; \frac{\sin x}{x} \;\le\; 1$$

Como $\cos x \to 1$ cuando $x \to 0$, por el Teorema del Sandwich: $\displaystyle\lim_{x \to 0} \frac{\sin x}{x} = 1$.
</Proof>
</Theorem>

<Theorem title="Identidades pitagĂłricas">
$$\sin^2 x + \cos^2 x = 1, \qquad \tan^2 x + 1 = \sec^2 x, \qquad 1 + \cot^2 x = \csc^2 x$$
</Theorem>

<Theorem title="FĂłrmulas de ĂĄngulo doble">

</Theorem>

<Theorem title="FĂłrmulas de suma de ĂĄngulos">
$$\sin(x + y) = \sin x \cos y + \cos x \sin y$$
$$\cos(x + y) = \cos x \cos y - \sin x \sin y$$
</Theorem>

<Theorem title="Identidades de reducciĂłn de potencias">
